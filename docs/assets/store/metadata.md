# Store Listing Assets

## App Name
MyTvRemote: Universal TV Control

## Short Description
Control every TV in your home with secure Wi-Fi, Bluetooth, and IR automation.

## Full Description
- Discover and control Samsung, LG, Roku, and Android TV devices in seconds.
- Customize command layouts, create automation scenes, and sync preferences across devices.
- Fallback to IR and HDMI-CEC bridges when network control is unavailable.

## Screenshots (Guidance)
1. **Home Dashboard:** Device grid with live status indicators.
2. **Device Control:** Command pad with customizable buttons.
3. **Scenes:** Automation editor with scheduling.
4. **Onboarding:** Permission prompts and discovery wizard.
5. **Settings:** Vendor integrations and privacy controls.

## Icon & Feature Graphic
- Icon: Stylized remote with tri-band signal waves (IR red, BLE blue, Wi-Fi green).
- Feature Graphic: Living room collage highlighting cross-brand compatibility.
